# 一，拿下主机后

当你拿下一台主机，如果这台主机上部署Java应用，可以使用jps 

```
ps -aux | grep java -jar //查看进程完后直接导出


jmap -dump:live,format=b,file=heapdump_18996.hprof 18996 //导出 java进程的 heapdump
```
